ATIVIDADE 01 - OPERAÇÕES
  1. Crie uma expressão com no mínimo 4 valores utilizando os operadores (*,+,/,-) retorne: 45, -30, 135, 90.

    20 + 40 * 2 - 55 = 45
    60 / 3 * 2 - 70 = -30
    100 * 2 + 10 - 75 = 135
    90 - 90 + 30 * 3 = 90

  2. Crie uma expressão com no mínimo 4 valores utilizando os operadores (*,+,/,-) e obrigatoriamente utilizando pelo menos um () e uma operação com módulo (%) retorne: 2, 50, -80, 10.

    (60 * 2 - 98) % 10 = 2
    (2050 - 1050 + 50) % 1000 = 50
    (-400 / 2 * 1) % 120 = -80
    (10 * 3 + 2) % 22= 10

  3. Crie uma expressão booleana que utilize pelo menos 3 operadores boleanos que retorne: true, false.

    ((3 == 3 && "a" == "g") && ("c" == "c" && 1 == 2)) = FALSE
    ((3 == 3 && "a" == "a") || ("c" != "c" || 100 == 100)) = TRUE
